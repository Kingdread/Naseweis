Overview

The Converter class provides a way to convert between stringy data and native Ruby types. It's used to handle the type: attribute of questions.

def convert(data, type)
  type = type.intern
  raise ArgumentError, "Invalid type #{type}" unless @converters.key? type
  begin
    @converters[type][data]
  rescue
    raise ConversionError.new data, type
  end
end
